# Brandon Hinrichs - Professional Resume Website

This repository contains the code for my professional online resume website. The site showcases my skills, work experience, education, and professional references.

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Clean, modern interface with smooth animations
- Interactive elements including skill level visualizations
- Contact form for potential employers or clients
- Print-friendly version for physical copies
- Consistent navigation across all pages

## Pages

1. **Home** - Overview of skills and professional summary
2. **Work History** - Detailed employment history with responsibilities
3. **Education & Certifications** - Academic background and professional certifications
4. **References** - Professional references with contact information

## Technologies Used

- HTML5
- CSS3 (with animations and responsive design)
- JavaScript (for interactivity)
- Bootstrap (for layout and components)
- Font Awesome (for icons)

## Setup and Deployment

To run this website locally:

1. Clone this repository
2. Open `index.html` in your web browser

For deployment:
- Upload all files to your web hosting service
- Ensure all paths are correctly set for your hosting environment
